### **변수 타입별 선언 방법**
①  정수형 변수 선언 방식 
```java
int 변수명 = 값;
```
<br/>

② 실수형 변수 선언 방식 
```java
float 변수명 = 값f;
```
<br/>

③ 실수형 변수 선언 방식 
```java
double 변수명 = 값;
```
<br/>

④ 문자형 변수 선언 방식 
```java
char 변수명 = '문자1개';    // char 타입은 문자 1개를 ''로 묶어 대입
```
<br/>

⑤ 문자열 변수 선언 방식 
```java
String 변수명 = "문자열";    // 문자열 타입은 ""로 묶어 대입
```
<br/>
