# exam04

### **자바 기본 문법 4 (반복문)**
for문, while문 예제 <br></br>

### **for 문(기본 형식)**
```java
for(초기식; 조건식; 증감식){
    // 해당 부분을 반복 실행 
}
```
<br/>

### **for 문(배열을 지원하는 형식-for each문)**
```java
for(변수형 변수 : 배열명){
    // 해당 부분에 변수 사용 
}
```
<br/>

### **while 문(기본 형식)**
```java
while(조건식) {
    // 조건식이 true인 동안 이 부분을 반복 실행 
}
```
<br/>

> while문 탈출 시 **break 문** 사용, 반복문의 조건식을 건너뛰려면 **continue문을** 사용
