### **기본적인 자바 main 클래스 구조**
①  클래스 정의 
```java
public class "클래스명" // 클래스명은 프로젝트 파일 이름과 동일해야합니다.
```
<br/>

② main 메소드 
```java
pulic static void main(String[] args) {
    //  어플리케이션이 실행되면 제일 먼저 메인(main)메소드를 실행한다. (RUN) 
}
```
<br/>

③ 출력 코드
```java
System.out.println("");   // 콘솔 창에 글자를 출력하는 코드
```
